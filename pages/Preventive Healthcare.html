簡介 INTRODUCTION

超預防醫學提供B2C/B2B的線上諮詢平台，致力於為世界各地的患者提供優質的醫療諮詢服務。透過與全台百家名醫合作，為平台提供全面的醫療評估、專業檢測及健康管理建議。這次的專案旨在打造一個直觀、高效的醫療預約網站，幫助患者及早預防潛在健康問題。

FEATURES
專案特色
# 百大名醫，資訊透明，專業服務安心問診

與全台醫院合作，邀請醫師線上問診，將每位醫師的背景與專業以卡片式呈現，方便使用者可以快速瀏覽。

同時串接衛福部資料，設計一份篩選表單，方便使用者依據價格、醫院、部位或疾病類型等項目快速查找適合自己的問診醫師。

# 增加裝置測試與監控，確保諮詢體驗流暢度

為確保諮詢體驗的流暢與穩定，進入諮詢室前增加裝置連線測試流程。參與者透過兩階段的連線測試來確認設備皆有開啟影音授權，以及網路環境是否順暢。當進入諮詢會議室後，畫面右上顯示網路、裝置授權、麥克風標誌，方便管理者能夠即時掌握與會者當前狀態並協助排除突發狀況。

# 標準化後台預約流程，資訊操作一目瞭然

以機票為靈感，將後台的預約管理編排成票卷形式，將不同類別的資料與功能區分方便管理人員能夠更直覺的操作。一張訂單管理卡片包含了10個以上 的功能操作，除了手動設定外，也配合系統自動化功能，根據不同情境下切換按鈕與選單的狀態，確保每位用戶的訂單都能完整的執行。

# 突破傳統限制，貼近使用者情境

打破以往後台表格的形式，超預防醫學嘗試將資料 集中成一張張卡片，避免表格需左右滾動的困擾， 能在同一畫面完整瀏覽所有資訊。在使用上更符合 流程的介面設計，不僅提升管理效率，同時也為平 台從啟用至今吸引超過300名以上的用戶預約。
